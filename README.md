# 🚀 TaskPilot – AI Powered Productivity Companion

TaskPilot is an AI-powered productivity companion designed to help users organize, prioritize, and complete tasks efficiently. Powered by Google Gemini AI, it goes beyond traditional task management by providing intelligent productivity suggestions and personalized assistance.

🌐 **Live Demo:** https://task-pilot-lake-pi.vercel.app/

🔗 **Backend API:** https://taskpilot-backend-8ax3.onrender.com/

---

## ✨ Features

- 🔐 User Authentication
- ✅ Create, Update & Delete Tasks
- 🤖 AI-powered Productivity Assistant (Google Gemini)
- 📅 Smart Task Planning
- 🎯 Focus Mode
- 📊 Interactive Dashboard
- 📱 Responsive User Interface
- ⚡ FastAPI Backend
- ⚛️ React + Vite Frontend

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SHREYA-03112006/TaskPilot.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **backend** folder.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```
